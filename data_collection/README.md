# Data Collection module for srl

* Chanyoung Ahn (cold-young)

* data_collection.py
   - You can collecting deformable objects point cloud data.
   
   - Examples
   ```bash
   python data_collection.py --object=tofu --num=10 --gravity --extract_indices
   ```

    - Can use various options
   ```
   headless, 
   data_num, 
   norm_pcd, 
   object, 
   gravity, 
   extract_indices, 
   vis_pcd
   ```
